# Contributing to BoozedBunnyTown

Thank you for your interest in BoozedBunnyTown! We are excited to build this browser game with the help of the community and modern technology.

## AI-Assisted Development Policy
This project embrace AI-assisted development (e.g., GitHub Copilot, ChatGPT, Claude). We encourage you to use these tools to generate code, refactor logic, or brainstorm features. 

However, please keep the following rules in mind:
* **Human Oversight:** Every AI-generated contribution must be reviewed and tested by a human.
* **Code Quality:** Ensure that AI-generated code adheres to our project structure and TypeScript standards.
* **No "Black Box" Logic:** You should be able to explain what the contributed code does, even if it was generated by an AI.

## How to Contribute

### 1. Find or Create an Issue
Before starting any work, please check the existing issues or create a new one to discuss your ideas. This prevents double work and ensures the feature aligns with the game's direction.

### 2. Workflow
1.  **Fork** the repository and create your branch from `main`.
2.  **Implement** your changes (AI assistance is welcome!).
3.  **Test** your changes locally. Ensure the build passes and the game logic works as expected.
4.  **Submit a Pull Request (PR)**:
    * Reference the related issue.
    * Describe what you changed.
    * Mention if you used AI tools for specific parts of the logic.

### 3. Human Review & Approval
All Pull Requests require a manual review and approval before being merged into the `main` branch. We focus on:
* Game balance and direction.
* Security and performance.
* Maintainability of the code.

## Tech Stack
We build on NodeJS. Detailed setup instructions can be found in the `README.md`.
